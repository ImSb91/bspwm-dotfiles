# Packages 

* `bspwm`: Window manager
* `dunst`: Notification daemon
* `nvim`: Editor
* `pfetch`: System info
* `picom`: Compositor
* `polybar`: Status bar
* `ranger`: File manager
* `rofi`: Application launcher
* `sxhkd`: Hotkey daemon
* `zathura`: Pdf reader
* `zsh`: Terminal emulator
* fonts:
   - ttf-jetbrains-mono
   - ttf-jetbrains-mono
   - nerd ttf-terminus-nerd
   - ttf-inconsolata ttf-joypixels
   - ttf-icomoon-feather
   - ttf-meslo-nerd-font-powerlevel10k
