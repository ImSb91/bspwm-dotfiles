# Packages 
* `zsh`: Terminal emulator
* `rofi`: Application launcher
* `ranger`: File manager
* `bspwm`: Window manager
* `sxhkd`: Hotkey daemon
* `dunst`: Notification daemon
* `polybar`: Status bar
* `zathura`: Pdf reader
* `picom`: Compositor
